# RATATOUILLE RUSH
Trabajo práctico final desarrollado en la materia de Fundamentos de la Programación - cátedra Mendez.

---

- Para compilar:

```bash
make
```
- Para jugar:

```bash
make jugar
```

---

## Introducción

Ratatouille Rush es un juego ambientado en el universo de *Ratatouille*, donde el jugador asume el rol del mozo principal del restaurante de Remy.  
El objetivo es atender a los clientes durante una jornada de trabajo, tomando pedidos, sirviendo platos y manteniendo el restaurante limpio antes del cierre de la cocina.

Cada nivel presenta una meta de dinero a alcanzar, poniendo a prueba la rapidez y eficiencia del jugador mientras el restaurante se vuelve cada vez más caótico.

---

## Objetivo

El objetivo de este trabajo práctico es aplicar y reforzar los siguientes conceptos:
- Desarrollo de funcionalidades respetando un contrato preestablecido.
- Uso de tipos de datos estructurados.
- Manejo de memoria dinámica y punteros.
- Desarrollo de una interfaz gráfica clara y amigable.
- Aplicación de buenas prácticas de programación como modularización, reutilización y claridad del código.

---

## Enunciado
Este proyecto corresponde a la **segunda parte** del trabajo práctico y constituye la versión final del juego.  
En esta etapa se completa la funcionalidad del juego, asignando comportamiento a los distintos elementos para que el juego sea completamente jugable.
